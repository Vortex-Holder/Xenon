# Xenon [ PROJECT DISAPLED ]
This terminal doesn't just have coding inside; there are surprises for trying it out.  People Playground is a surprise as a game.  And many more things!
## Permissions
You cannot copy this terminal without permission and present it as your own, otherwise you will be added to the blacklist.
However, you can only use this feature with permission and you must state the reason.
## Features
* Easter Eggs: Hidden surprises (like the games we talked about).
* Safe Payloads: MEMZ and ILOVEYOU visual simulations.
* ConEmu Core: Built on the powerful ConEmu engine for a high-performance terminal experience.
## About
### Why Xenon?
* Not just a terminal, but a workspace for developers and enthusiasts.
* Focused on aesthetics and functionality.

## System Requirements
* OS: Windows 7 / 8 / 10 / 11 (Optimized for low-end PCs).
* RAM: 2GB minimum (Works great on 4GB+).
* Storage: ~6GB (Including the tools and games).
